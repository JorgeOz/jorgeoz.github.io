# 🌐 Mi Porfolio Personal - JorgeOz

Este repositorio contiene el código fuente de mi porfolio personal, desplegado a través de **GitHub Pages**.

## 🛠️ Tecnologías utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Diseño personalizado (estilo oscuro) con variables y Flexbox/Grid.
* **Google Fonts:** Tipografías 'Inter' y 'Fira Code'.
